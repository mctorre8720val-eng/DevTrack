# DevTrack

## Overview

DevTrack is a project management application.

It allows users to create projects, manage tasks, and track progress.

The goal of DevTrack is to create a real-world application while learning React development, Firebase integration, and AI-assisted software development.

## Goals

- Learn React
- Practice Firebase
- Learn proper project structure
- Build a production-style application
- Learn AI agent workflow

## Tech Stack

Frontend:
- React
- Vite
- React Router

Backend:
- Firebase

Database:
- Firestore

## Features

- User authentication
- User profiles
- Project creation
- Project editing
- Project deletion
- Task management
- Task completion tracking
- Dashboard
